# Heyy

Hey is an open source next gen translation app for humans

## Prequesites

Create `env.local` and populate with the following variables

### 1. Clerk

```sh
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
CLERK_SECRET_KEY=
```

### 2. Google Translate API

## Getting Started

npm run dev
